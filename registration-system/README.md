# ระบบลงทะเบียน (Registration System)

ระบบรับลงทะเบียนที่เก็บ **ชื่อ / อีเมล / เบอร์โทร** ไว้ที่เซิร์ฟเวอร์ (ไฟล์ฝั่ง backend)
พร้อมหน้าหลังบ้านสำหรับดูรายชื่อผู้ลงทะเบียน สร้างด้วย Node.js ล้วน ไม่ต้องติดตั้ง package เพิ่ม

## วิธีรัน

```bash
cd registration-system
node server.js
```

จากนั้นเปิดเบราว์เซอร์:

- หน้าลงทะเบียน: http://localhost:3000
- หน้าหลังบ้าน:  http://localhost:3000/admin

## รหัสผ่านหน้าหลังบ้าน

ค่าเริ่มต้นคือ `admin123` — เปลี่ยนได้โดยกำหนด environment variable ก่อนรัน:

```bash
# Windows (PowerShell)
$env:ADMIN_KEY="รหัสใหม่"; node server.js

# macOS / Linux
ADMIN_KEY="รหัสใหม่" node server.js
```

## ข้อมูลถูกเก็บที่ไหน

`data/registrations.json` (สร้างอัตโนมัติเมื่อรันครั้งแรก) — เก็บทุกครั้งที่มีคนลงทะเบียน
ข้อมูลจึงอยู่แม้ปิดเซิร์ฟเวอร์แล้วเปิดใหม่

## ฟีเจอร์

- ✅ ฟอร์มลงทะเบียนพร้อมตรวจสอบข้อมูล (email รูปแบบถูกต้อง, เบอร์ 9-10 หลัก)
- ✅ กันลงทะเบียนอีเมลซ้ำ
- ✅ หน้าหลังบ้านมีรหัสผ่าน ดูรายชื่อทั้งหมด + ค้นหา
- ✅ ลบรายการทีละคนได้
- ✅ ดาวน์โหลดรายชื่อเป็น CSV (เปิดใน Excel ได้ รองรับภาษาไทย)

## หมายเหตุด้านความปลอดภัย

ระบบนี้เหมาะกับใช้งานภายใน / งานเล็ก การป้องกันหน้าหลังบ้านเป็นรหัสผ่านแบบง่าย
หากต้องเปิดใช้บนอินเทอร์เน็ตจริง ควรเพิ่ม HTTPS, ระบบล็อกอินที่แข็งแรงขึ้น และฐานข้อมูลจริง
